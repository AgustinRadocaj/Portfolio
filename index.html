Proyectos

PI Videogames

Proyecto Fullstack con pedidos a una API externa. La finalidad del proyecto era la muestra de un catalogo de videojuegos. Con la posibilidad de buscar por nombre y aplicar filtros a la busqueda, tambien se pueden ingresar videojuegos de manera manual y agregarlos a una base de datos para que puedan ser mostrados junto con todo el catalogo disponible. Por ultimo se puede ingresar a cada juego para ver su informacion mas detallada.

Features

  • Pedidos a API
  • Filtro de ordenamiento
  • Creacion y almacenamiento de juegos con base de datos
  • Barra de busqueda por nombre
Tecnologias: ReactJS - Express.js - SQL

StayFi

StayFi es un emocionante proyecto de ecommerce de libros que ofrece a los usuarios una plataforma conveniente y segura para explorar y comprar una amplia gama de títulos. Con características de usuario y administrador, implementación de borrado lógico, metodología Scrum y una pasarela de pago integrada, nuestro proyecto se esfuerza por brindar una experiencia fluida y segura

Features

  • Login, registro y restaracion de contraseña
  • Pasarela de pagos (Mercado Pago)
  • Dashboard de administrador
  • Almacenamiento en la nube (Cloudinary)
Tecnologias: ReactJS - Tailwind - Express.js - SQL
